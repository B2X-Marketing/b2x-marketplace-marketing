# Report Template Reference

The full markdown report template is at `Research/Templates/report-template.md`.

When generating the report, follow these rules:

## Structure
1. Header block with metadata (date, mode, company, competitors, confidence)
2. Executive Summary (written LAST, after all dimensions)
3. Six numbered sections, one per dimension
4. Sources section at the end, grouped by dimension

## Writing Rules
- Follow the voice profile at `Voice_Profile_Luis_Capobianco.md`
- Active voice, lead with the point
- No "delve", no filler transitions ("Furthermore", "Additionally"), no excessive em dashes
- Numbers as digits (not words)
- Short, direct sentences
- Present data confidently but flag uncertainty honestly
- Every data point must cite its source

## Section Rules
- Each section starts with a 2-3 sentence **Overview** summarizing the dimension
- Use tables for structured comparisons (competitors, features, trends, pain points)
- Use bullet lists for strengths/weaknesses/differentiators
- Use numbered lists for ranked items (decision factors, recommendations)

## Executive Summary Rules
- Write 3-5 paragraphs covering all six dimensions
- Lead with the most important finding
- Include key numbers: market size, growth rate, number of competitors
- End with the top opportunity and a forward-looking statement
- Do NOT repeat section summaries verbatim — synthesize across dimensions

## Source Citation Rules
- Group sources by dimension at the end of the report
- Format: `[Title](URL) — domain, accessed YYYY-MM-DD`
- Number sources sequentially within each dimension group
- In-text references: use the source name, not a numbered citation
