# Framework Overview

## The Problem

No single marketing framework covers everything a company needs. The Golden Circle defines purpose but not tactics. RACE maps the customer journey but not messaging. StoryBrand nails communication but not channel strategy. SOSTAC plans the operation but not the value proposition.

## The Solution: An 8-Layer Stack

This framework layers eight complementary marketing models. Each layer handles what it does best and produces artifacts that the next layer consumes.

```
┌─────────────────────────────────────────────────────┐
│ Layer 1: GOLDEN CIRCLE (Sinek)                      │
│ Purpose & Identity → WHY / HOW / WHAT               │
├─────────────────────────────────────────────────────┤
│ Layer 2: CUSTOMER INSIGHT (JTBD + VPC + STP)        │
│ Who you serve → Segments, Jobs, Value Props          │
├─────────────────────────────────────────────────────┤
│ Layer 3: STORYBRAND SB7 (Miller)                    │
│ How you communicate → Brand Scripts, One-Liners      │
╞═════════════════════════════════════════════════════╡
│         ─── LEAD-GEN BOUNDARY ───                   │
│  (Layers 1-3 form a self-contained deliverable)     │
╞═════════════════════════════════════════════════════╡
│ Layer 4: SOSTAC (PR Smith)                          │
│ Strategic Planning → Situation thru Control          │
├─────────────────────────────────────────────────────┤
│ Layer 5: RACE (Chaffey)                             │
│ Customer Journey → Reach / Act / Convert / Engage    │
├─────────────────────────────────────────────────────┤
│ Layer 6: PESO (Dietrich)                            │
│ Channel & Media → Paid / Earned / Shared / Owned     │
├─────────────────────────────────────────────────────┤
│ Layer 7: CONTENT MARKETING MATRIX                   │
│ Content Plan → Quadrant mapping + 90-day calendar    │
├─────────────────────────────────────────────────────┤
│ Layer 8: P&G WHO/WHAT/HOW  [B2C/Both only]           │
│ Validation → Cross-check against classic framework   │
│ (B2B → skip Layer 8, go directly to Synthesis)       │
└─────────────────────────────────────────────────────┘
```

## Layer Dependencies

Each layer consumes outputs from earlier layers:

- **Layer 1** → standalone (no dependencies)
- **Layer 2** → requires Layer 1 (WHY/HOW/WHAT inform segmentation and positioning)
- **Layer 3** → requires Layers 1-2 (segments become heroes, JTBD become problems, WHY becomes guide authority)
- **Layer 4** → requires Layers 1-3 (all identity/insight/messaging feeds into strategic planning)
- **Layer 5** → requires Layer 4 (objectives and tactics map into RACE stages)
- **Layer 6** → requires Layer 5 (RACE tactics need channel assignments)
- **Layer 7** → requires Layers 3, 5, 6 (brand script + journey stages + channels = content plan)
- **Layer 8** → requires all above (validation cross-check) — **consumer/retail brands only**

**Rule**: Always complete layers in order. Skipping creates downstream gaps.

**Layer 8 Routing**: The WHO-WHAT-HOW validation (Layer 8) was designed for consumer packaged goods and retail brands. It does not apply to B2B engagements. Route as follows:
- **B2B**: Layer 7 → Synthesis (skip Layer 8; the Synthesis coherence checklist absorbs its validation value)
- **B2C or consumer-facing**: Layer 7 → Layer 8 → Synthesis
- **Both (B2B + B2C)**: Run Layer 8 for consumer-facing segments only, then proceed to Synthesis

## Scenario Routing

Not every engagement needs the same emphasis. The intake process (file 01) classifies the scenario:

| Scenario | When to Use | Layer Modifications |
|----------|------------|-------------------|
| **NEW_COMPANY** | Startup or pre-launch, no existing strategy | Full depth on all layers |
| **EXISTING_PIVOT** | Company changing direction, rebranding, new market | Heavy emphasis on Layers 1-2 (redefining purpose and audience) |
| **PRODUCT_LAUNCH** | New product/service within established company | Light Layer 1 (inherit company WHY), heavy Layers 2-3 and 5-7 |
| **STRATEGY_REFRESH** | Existing strategy needs updating | Start with Layer 4 Situation Analysis, backfill gaps in earlier layers |

## Time Estimates

| Layer | Typical Duration | Notes |
|-------|-----------------|-------|
| Intake | 10-15 min | Company basics and scenario classification |
| Layer 1: Golden Circle | 20-30 min | Foundational — worth spending time here |
| Layer 2: Customer Insight | 30-45 min | Most complex layer (three sub-frameworks) |
| Layer 3: StoryBrand | 20-30 min | Per primary segment |
| Layer 4: SOSTAC | 30-45 min | Depends on available data |
| Layer 5: RACE | 20-30 min | Per primary segment |
| Layer 6: PESO | 20-30 min | Channel selection and allocation |
| Layer 7: Content Matrix | 20-30 min | Content planning and calendar |
| Layer 8: P&G Validation | 10-15 min | Cross-check exercise |
| Synthesis | 20-30 min | Final assembly and review |
| **Total** | **3-5 hours** | **Can be split across multiple sessions** |

## Multi-Session Support

Sessions can be paused and resumed at any layer boundary. The strategy output document contains a "Strategy State" section that tracks:
- Current scenario classification
- Which layers are complete
- Key decisions made so far (WHY statement, target segments, etc.)
- What layer comes next

When resuming, Claude reads the output document, checks the Strategy State, recaps prior work, and picks up where the user left off.
